Цель: создать начальный словарь из 2000 наиболее частотных
английских слов с переводами. В дальнейшем использовать его, как базовый
для WordCatcher и как изучаемый и расширяемый для MyAnki

Процедура:

Считываем первые 2000 из 5000_Frequent. Это основа для словаря
именно в таком порядке.

Считываем 2000_Translated
Считываем 5000 самых употребительных английских слов (преобразованный в txt)
Создаем итоговый словарь. При этом учитываем возможность неоднократного входа
одного и того же слова.

Создаем словарь для записи. Key - из 5000_Frequent (2000), 
учесть возможность повторения слов (хотя вряд ли)

Value - из 5000_Frequent & 5000 самых употребительных. 
Записываем все несовпадающие переводы. 

Если нет ни одного перевода - в файл unknown words.

Результат - словарь, подходящий для конвертации в MyAnki (DictConvert)
и для работы WordCatcher.
